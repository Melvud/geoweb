# Как запустить сайт у себя

Эта инструкция для запуска проекта `GEOweb` на Windows.

## 1. Установить Node.js

Откройте сайт:

https://nodejs.org

Скачайте версию **LTS** и установите ее.

Во время установки ничего особенного менять не нужно, просто нажимайте **Next**.

После установки лучше перезагрузить компьютер. Если не хочется перезагружать, закройте все окна PowerShell/Terminal и откройте заново.

## 2. Распаковать проект

Распакуйте архив с проектом, например сюда:

```text
C:\Users\ИмяПользователя\Documents\GEOweb
```

Внутри папки `GEOweb` должны быть файлы и папки:

```text
package.json
src
public
data
```

## 3. Открыть PowerShell в папке проекта

Откройте папку `GEOweb`.

Зажмите **Shift**, нажмите правой кнопкой мыши в пустом месте папки и выберите:

```text
Открыть в терминале
```

или:

```text
Open PowerShell window here
```

Если PowerShell открыт не в той папке, можно перейти вручную:

```powershell
cd "C:\Users\ИмяПользователя\Documents\GEOweb"
```

Путь нужно заменить на свой, если проект распакован в другое место.

## 4. Проверить Node.js и npm

В PowerShell выполните:

```powershell
node -v
npm -v
```

Если появились версии, например `v22...` и `10...`, все нормально.

Если пишет, что `node` или `npm` не найден, значит Node.js не установлен или PowerShell был открыт до установки. Закройте PowerShell, откройте заново и попробуйте еще раз.

## 5. Установить зависимости

В папке проекта выполните:

```powershell
npm install
```

Это может занять несколько минут.

## 6. Запустить сайт

После установки зависимостей выполните:

```powershell
npm run dev
```

Если все хорошо, появится строка вроде:

```text
Local: http://localhost:3000
```

## 7. Открыть сайт

Откройте в браузере:

```text
http://localhost:3000
```

Админка:

```text
http://localhost:3000/admin
```

## 8. Как остановить сайт

В окне PowerShell, где запущен сайт, нажмите:

```text
Ctrl + C
```

Если появится вопрос, подтвердите остановку.

## 9. Как запустить сайт в следующий раз

Во второй и последующие разы `npm install` обычно делать не нужно.

Достаточно открыть PowerShell в папке проекта и выполнить:

```powershell
npm run dev
```

Потом открыть:

```text
http://localhost:3000
```

## Если что-то не работает

### `npm` не найден

Установите Node.js LTS с сайта:

https://nodejs.org

Потом закройте и снова откройте PowerShell.

### Порт 3000 занят

Если написано, что порт `3000` занят, закройте старое окно PowerShell, где уже мог быть запущен сайт.

Если не помогает, перезагрузите компьютер и попробуйте снова.

### Страница не открывается

Проверьте, что команда `npm run dev` все еще работает и окно PowerShell не закрыто.

Сайт работает только пока запущена эта команда.
